# aleabow
Beta inspiration from hanenbow

click aleabow(v3).html under the read me 

download the raw file 

from download folder, click and drag the .html file into a chrome browser (p5.js only works on chromium)

____________________

Hardware MIDI synths will be detected immediately 

If using DAW, particuluar Mac OS, ensure your IAC bus driver is activated. It's a virtual midi bus. 

____________________

It's rudimentary and simple, a passion project if you will. 

Rate and Speed are kinda funky, you'll want to increase the speed to start. 

The two purple walls help bounce the "fish balls" around. Careful to cross their paths because there's some magnetic issue I have to debug lol. 

Add fx to your synth and enjoy some aleatoric music processing!
